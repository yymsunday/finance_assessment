金融数据挖掘实训系统
一个基于 Streamlit 的交互式金融数据分析平台，用于股票数据可视化、行业分析、综合评价与量化策略回测。

📌 功能概览
市场总览：查看主要指数行情、龙虎榜涨跌统计、行业财务数据排名

行业分析：按申万一级行业筛选股票，查看行业走势、个股表现、财务与交易数据

综合评价：结合财务数据与股价收益进行综合评分与排名

量化策略：技术指标计算（MA、RSI）、机器学习模型演示、策略回测

🗂 文件结构
text
金融数据挖掘.py          # 主程序
复权交易数据2023.parquet   # 股票交易数据（2023）
复权交易数据2024.parquet   # 股票交易数据（2024）
复权交易数据2025.parquet   # 股票交易数据（2025）
最新个股申万行业分类(完整版-截至7月末).xlsx   # 行业分类数据
fin_data.csv               # 财务数据
沪深300指数交易数据.xlsx   # 沪深300指数数据（可选）
🚀 快速开始
1. 安装依赖
bash
pip install streamlit pandas numpy plotly openpyxl
2. 准备数据
将以下文件放在同一目录下：

至少一个 复权交易数据20XX.parquet

最新个股申万行业分类(完整版-截至7月末).xlsx

fin_data.csv

（可选）沪深300指数交易数据.xlsx

3. 运行应用
bash
streamlit run 金融数据挖掘.py
🧩 主要模块说明
数据加载
load_all_adj_data()：加载多个年份的复权交易数据

load_industry_class()：加载行业分类数据

load_fin_data()：加载财务数据

load_hs300_from_excel()：加载沪深300指数数据

核心功能
指数走势图：显示上证A股、深证A股、沪深300指数走势

龙虎榜统计：筛选涨跌幅超过±20%的股票

行业统计：按行业汇总营业收入、利润、公司数量

综合评价：结合净利润与持有期收益率进行排名

量化策略：计算MA、RSI指标，演示模型训练与预测流程

📊 数据说明
交易数据：包含 ts_code（股票代码）、date（日期）、close（收盘价）

行业数据：包含股票代码与申万一级行业分类

财务数据：包含营业收入、净利润、ROE、ROA等指标

⚙️ 配置说明
默认时间范围：2023年1月1日 - 2023年12月31日

行业分类基于申万一级行业标准

沪深300数据优先从独立文件加载，若无则从交易数据中提取

🧪 使用示例
启动应用后，在侧边栏选择"市场总览"或具体行业

查看指数走势、行业排名、个股表现

在"综合评价分析"中调整持有期和排名数量

在"技术指标与量化策略"中查看指标计算和策略回测结果

📈 注意事项
若无沪深300独立文件，系统会自动从交易数据中查找 399300.SZ 或 000300.SH

行业指数走势由行业内个股平均价格近似计算

财务数据单位为亿元，收益率单位为百分比
